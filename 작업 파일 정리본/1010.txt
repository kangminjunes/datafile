Linux 운영체제 - 주로 서버에 설치된 운영체제
프로젝트 결과물을 서버에 올리기 위해서 미리 사용해 보기

1. Linux 운영체제를 설치할 가상 컴퓨터 - Virtual Box
2. 가상 컴퓨터에 Linux 운영체제 설치하기
3. 가상 서버(가상 컴퓨터 + Linux 운영체제)에 접속 - SSH (PuTTY)

Linux
- Debian : Ubuntu
- RedHat : Centos
- SuSe

- 호스트 컴퓨터 : 192.168.56.1
- 게스트 컴퓨터 : 10.0.2.15
